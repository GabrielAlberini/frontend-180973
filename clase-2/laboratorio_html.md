# Laboratorio HTML – Construcción de una página web explorando etiquetas

## Consigna

Vas a crear una página web usando únicamente HTML.  
El objetivo es aprender explorando: vas a usar ChatGPT como asistente para descubrir etiquetas y probarlas en tu propio proyecto. No se busca memorizar, sino experimentar y comprender cómo se estructura una página web.

---

## Parte 1: Estructura base obligatoria

Creá un archivo `index.html` en Visual Studio Code con la siguiente estructura:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Mi laboratorio HTML</title>
</head>
<body>

</body>
</html>
```

Todo el contenido de tu página debe estar dentro del `<body>`.

---

## Parte 2: Investigación guiada con ChatGPT

Usá ChatGPT con la siguiente consigna:

> "Actuá como profesor de HTML para principiantes. Necesito aprender etiquetas HTML básicas. Dame entre 15 y 20 etiquetas distintas explicadas de forma simple, con un ejemplo corto de cada una. Agrupalas por categorías si es posible (texto, estructura, multimedia, listas, enlaces)."

---

## Parte 3: Construcción de la página

Con la información obtenida, construí una página que incluya como mínimo:

### Estructura y texto

- Un título principal (`h1`)
- Dos subtítulos (`h2` o `h3`)
- Al menos 3 párrafos con contenido distinto

### Organización del contenido

- Una lista desordenada (`ul`)
- Una lista ordenada (`ol`)
- Una sección o bloque separado usando una etiqueta estructural como `section` o `div`

### Interacción básica

- 2 enlaces (`a`) a sitios externos
- 1 enlace que abra en una nueva pestaña

### Multimedia

- 1 imagen (`img`)
- 1 elemento multimedia adicional (puede ser `video` o `audio`). Si no lo utilizás, agregá otra imagen con contenido distinto

### Exploración de etiquetas

- Agregá al menos 5 etiquetas adicionales descubiertas con ChatGPT
- Podés usar etiquetas como `strong`, `em`, `mark`, `br`, `hr`, `blockquote`, `code`, `pre`, `small`, entre otras

---

## Parte 4: Modo explorador

Elegí al menos:

- 2 etiquetas que no entiendas
- 1 etiqueta poco común o desconocida

Consultá a ChatGPT:

> "¿Qué hace la etiqueta `<X>` en HTML? Dame un ejemplo simple y cuándo se usa."

Luego incorporalas a tu página aunque sea de forma experimental.

---

## Parte 5: Desafío creativo

Transformá tu página en una de estas ideas o en otra similar:

- Página personal
- Museo de cosas favoritas
- Diario personal
- Página de un videojuego
- Catálogo de películas o música

---

## Parte 6: Reflexión final

Al final del archivo respondé:

- ¿Qué etiqueta te resultó más útil?
- ¿Qué etiqueta te generó más dudas?
- ¿Qué descubriste sobre HTML?
- ¿ChatGPT te ayudó a entender o solo a copiar?

---

## Objetivo pedagógico

- Comprender HTML como lenguaje de estructura
- Explorar etiquetas mediante investigación guiada
- Construir una página real desde cero
- Usar ChatGPT como herramienta de aprendizaje y no solo de resolución automática
